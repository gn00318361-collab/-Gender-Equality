## 🎬 影片推薦簡報

每份簡報共 6 頁，推薦 3 部 YouTube 影片，每部影片附有可點擊的超連結按鈕。

**簡報結構：**

1. 封面頁（年段名稱與主題）
2. 推薦影片總覽（三部影片一覽 + YouTube 連結）
3. ~ 5. 各影片詳細頁（影片簡介、教學重點、課堂引導問題、延伸活動）
6. 教學小提醒（營造安全環境、鼓勵討論、連結生活經驗、提供求助資源）

## 🎮 互動網頁遊戲

每個年段各有一個 HTML 互動遊戲，**不需安裝任何軟體**，用瀏覽器開啟即可使用。支援電腦與手機。

### 使用方式

1. 下載 `.html` 檔案
2. 用瀏覽器（Chrome / Edge / Safari）直接開啟
3. 學生即可開始闖關遊戲

### 遊戲特色

| 年段 | 遊戲名稱 | 關卡 | 玩法 | 遊戲連結 |
|------|---------|------|------|---------|
| 低年級 | 性別平等小勇士 🛡️ | 8 關 | 選擇題 + 拖拉配對 | [開始遊戲](Gender_Equality_Game_Lower_Grades.html) |
| 中年級 | 性別平等大挑戰 🤝 | 8 關 | 選擇題 + OK/不OK 判斷 + 步驟排序 | [開始遊戲](Gender_Equality_Game_Middle_Grades.html) |
| 高年級 | 打破框架！💜 | 8 關 | 選擇題 + 觀點辯論 + 多選題 | [開始遊戲](Gender_Equality_Game_Upper_Grades.html) |

**遊戲機制：** 生命值（3 顆愛心）、連擊 Combo 加分、倒數計時、音效回饋、答對撒花特效

**低年級** — 涵蓋：身體自主權、隱私部位、勇敢說不、113 保護專線、情緒管理

**中年級** — 涵蓋：身體界線、認識性騷擾、同理心、網路騷擾

**高年級** — 涵蓋：性別刻板印象、玫瑰少年事件、迷思破解、情感教育、性別平等教育法

## 🏫 建議教學流程

```
課前準備
  └─ 選擇適合年段的簡報與遊戲

課堂教學（約 40 分鐘）
  ├─ 1. 開場引導（5 分鐘）
  │     使用簡報封面與總覽頁介紹今天的主題
  ├─ 2. 影片觀賞（10-15 分鐘）
  │     播放推薦影片，可選擇 1-2 部觀賞
  ├─ 3. 討論與分享（10 分鐘）
  │     使用簡報中的「課堂引導問題」進行討論
  ├─ 4. 互動遊戲（10 分鐘）
  │     讓學生使用平板或電腦進行闖關遊戲
  └─ 5. 總結（5 分鐘）
        使用簡報最後一頁的「教學小提醒」做總結
```

---

## ⚠️ 教學注意事項

- 播放影片前，提醒學生這是重要但有時會讓人不舒服的話題，讓學生知道有任何不舒服可以說出來
- 使用開放式問題引導討論，避免批判性語言，尊重每位學生的想法
- 如有學生在課後表達曾經歷相關事件，請引導至輔導室或撥打 **113 保護專線**
